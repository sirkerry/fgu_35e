# 3.5E/PFRPG ADV/DIS & Modifier Buttons

FGU extension for the official 3.5E (Dungeons and Dragons 3.5E) and PFRPG (Pathfinder 1E) rulesets.

Adds 5E-style modifier stack buttons — ADV/DIS, +2/-2, +5/-5 — to the desktop modifier stack panel.

**Requires "Feature: Extended Automation" to be loaded alongside this extension** — it supplies the underlying Advantage/Disadvantage dice mechanic; this extension supplies the missing buttons for it.

## How to Use

1. Enable both this extension and "Feature: Extended Automation," then load a 3.5E or PFRPG campaign.
2. Open the desktop modifier stack panel — it now shows **ADV**/**DIS**, **+2**/**-2**, and **+5**/**-5** buttons.
3. Click a button (or drag it to a hotkey) before making a roll, then roll any attack, save, skill, or initiative check as normal.
4. ADV/DIS roll the die twice and keep the favorable result; +2/-2/+5/-5 apply a flat modifier to the roll total.

## Compatibility

- Official 3.5E (Dungeons and Dragons 3.5E) and PFRPG (Pathfinder 1E) rulesets
- Requires "Feature: Extended Automation" to be loaded — without it, ADV/DIS buttons will click but have no effect on the dice
