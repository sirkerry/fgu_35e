# 3.5E/PFRPG Subraces

FGU extension for the official 3.5E (Dungeons and Dragons 3.5E) and PFRPG (Pathfinder 1E) rulesets.

Adds a Subraces section to the Race record. Applying a race with subraces to a character prompts for which one (auto-picked if there's only one), then applies that subrace's own traits alongside the base race's.

## How to Use

1. Open a Race record with the extension loaded — it now has a **Subraces** section above Traits.
2. Ctrl-drag another race's shortcut onto the Subraces header to copy it in as a new subrace (it gets its own Traits section to edit).
3. Apply the race to a character as normal. If it has subraces, you'll be prompted to pick one (skipped automatically if there's only one) — that subrace's traits are then applied alongside the base race's, and the character's race name updates to reflect the choice (e.g. "Dwarf" → "Mountain Dwarf", or "Elf (Grey)").

## Compatibility

- Official 3.5E (Dungeons and Dragons 3.5E) and PFRPG (Pathfinder 1E) rulesets
- Purely additive — doesn't remove or replace any stock Race functionality
